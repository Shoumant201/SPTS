# SPTM Passenger App

## New Dashboard Features

The passenger app now includes a modern dashboard interface based on the provided UI design:

### 🏠 Home Screen Features

- **Personalized Greeting**: Shows user's name and location
- **Destination Search**: Quick search bar for finding routes
- **Quick Actions**:
  - **Tap In/Out**: NFC-based payment system (primary action)
  - **Live Map**: Real-time bus tracking
  - **Wallet**: Digital payment management
  - **History**: Trip history and receipts

### 🚌 Nearest Bus Information

- **Real-time Arrival**: Shows next bus arrival time
- **Bus Details**: Bus number, route, and status
- **Live Tracking**: Track bus location in real-time
- **Route Information**: Destination and stops

### 📱 Bottom Navigation

- **Home**: Main dashboard (current)
- **Routes**: Browse all available routes
- **Tickets**: Manage tickets and passes
- **Account**: Profile and settings

### 🎨 Design Features

- Clean, modern interface matching the provided design
- Intuitive navigation with clear visual hierarchy
- Responsive layout optimized for mobile devices
- Consistent color scheme and typography

## Getting Started

```bash
npm install
npm start
```

The app uses phone-based authentication with OTP verification for secure access.
