# SPTM Driver App

## New Dashboard Features

The driver app now includes a professional dashboard interface based on the provided UI design:

### 👨‍💼 Driver Profile Header

- **Driver Information**: Name, ID, and profile picture
- **Status Indicator**: Active/Inactive status with visual indicator
- **Professional Layout**: Clean, organized driver information display

### 🗺️ Live Route Map

- **Interactive Map**: Shows current route and location
- **Live Route Button**: Quick access to live tracking
- **Visual Route Display**: Clear route visualization

### 🚌 Current Assignment

- **Bus Assignment**: Shows assigned bus number (e.g., Bus #402)
- **Route Information**: Current route details (e.g., Line 5 - North Shore)
- **Assignment Status**: Clear status indicators

### ⏰ Shift Management

- **Start Shift**: Large, prominent button to begin work
- **End Shift**: Easy shift termination
- **Shift Status**: Clear indication of current shift status

### ⚡ Quick Actions

- **Report Issue**: Emergency and incident reporting (red indicator)
- **View Earnings**: Track daily/weekly earnings (blue indicator)
- **Schedule**: View and manage work schedule
- **Dispatch**: Communication with dispatch center

### 📱 Bottom Navigation

- **Home**: Main dashboard (current)
- **Map**: Detailed route and navigation view
- **Stats**: Performance and earnings statistics
- **Profile**: Driver profile and settings

### 🎨 Design Features

- Professional driver-focused interface
- Color-coded action buttons for quick identification
- Clear visual hierarchy for important information
- Optimized for one-handed operation while driving
